# Paraclete

## Cluster Claim
This cluster groups the Johannine Paraclete passages that the working corpus reads as anticipating a later human messenger rather than exclusively the Holy Spirit.

## Main Use
- Hold the full Paraclete debate in one place instead of scattering it across John 14-16.
- Distinguish the departure condition, public mission, and mediated-speech arguments.
- Keep John 14:26 visible as the main contested verse.

## Objections / Alternate Readings
- The default Christian reading remains the Holy Spirit.
- Several supporting claims in the working files rely on textual criticism, variant readings, or source theories that need external verification before being treated as firm conclusions.
