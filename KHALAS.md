# KHALAS

## What This File Is
This is the project's current findings ledger. It does not repeat raw corpus text. It records what the repository currently treats as the main findings after normalization, passage classification, and convergence review.

## Status Legend
- `High-convergence`: central finding with strong corpus recurrence and stable argumentative role
- `Medium-convergence`: real finding, but more dependent on supporting chains or thinner recurrence
- `Control`: not a primary prophecy claim; used to test or pressure competing readings
- `Tentative`: present in the corpus but still too unstable to carry much project weight

## Notes on Probability
- "High-convergence" in this file means the finding is central inside the current corpus and repository structure, not that debate is over.
- "Medium-convergence" usually marks passages that need stronger supporting chains, more lexical care, or a larger inferential move.
- "Control" passages matter because they shape how the main passages are read; they are not mere footnotes.
